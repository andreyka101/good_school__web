# Good School Web Platform 🎓

Веб-платформа для автоматизации процессов онлайн-школы, личных кабинетов учеников и преподавателей, а также системы интерактивного расписания. 

Проект реализован как **Multi-Page Application (MPA)** на современном стеке с использованием строгой типизации, что обеспечивает высокую производительность, мгновенную загрузку страниц и независимость модулей.

---

## 🛠️ Стек технологий
* **Frontend Core:** HTML5, TypeScript (строгая типизация данных)
* **Сборщик проекта:** Vite (оптимизация сборки, быстрый Hot Module Replacement)
* **Стилизация:** SCSS / SASS (модульная структура, переменные, миксины)
* **Архитектура:** Multi-Page Application (MPA) для оптимизации SEO и разделения бизнес-логики

---

## 🚀 Ключевой функционал платформы
* **Личные кабинеты (Ролевая модель):**
  * `personal-area-student.html` — дашборд студента, история оплат, доступ к курсам.
  * `personal-area-teacher.html` — панель управления для преподавателя.
* **Интерактивное расписание:** 
  * Модули `setting-up-timetable` для гибкой настройки графиков занятий со стороны учителей и выбора слотов студентами.
* **Финансовый блок:**
  * Интеграция интерфейсов оплаты (`payment-page.html`), верификации транзакций и выбора тарифных планов/подписок.
* **Авторизация и безопасность:**
  * Экраны регистрации, восстановления и подтверждения паролей с валидацией полей на стороне клиента с помощью TypeScript.

---

## 📁 Архитектурная структура проекта
* `src/` — основная логика приложения, TypeScript-скрипты, модули инициализации компонентов.
* `articles/` & `blog/` — статические разделы контент-маркетинга платформы.
* Корневые `.html` файлы — изолированные модули страниц MPA-приложения для деления зон ответственности.
